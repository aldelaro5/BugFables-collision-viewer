After installing BepInEx, place the following dlls in this directory to build this plugin:

[Bug Fables]\Bug Fables_Data\Managed\Assembly-CSharp
[Bug Fables]\Bug Fables_Data\Managed\Assembly-CSharp-firstpass
[Bug Fables]\BepInEx\core\BepInEx
[Bug Fables]\BepInEx\core\BepInEx.Harmony
[Bug Fables]\BepInEx\core\0Harmony

Where [Bug Fables] is the directory where the game is installed.
If you are using Steam, it is by default located at 
C:\Program Files (x86)\Steam\steamapps\common\Bug Fables

Additionally, you will need to place the Unity 2018.4 DLLs in this directory. These DLLs are publically accessible after installing Unity. For your convenience, they are proviced in this link: https://drive.google.com/file/d/1e_pTXyCJDZLOhK67AIEVEMoZMPALfmL_/view?usp=sharing
